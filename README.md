# Filanwaa — Diinta, Tecnolijiyada iyo Suugaanta

Website caalami ah oo isku daraya Diinta Islaamka, Tecnolijiyada, iyo Suugaanta Soomaaliyeed. Lagu helo sedex luqadood: Somali, English, iyo Carabi.

## Features

- **3 Luqadood**: Somali, English, Carabi (RTL support)
- **3 Cinwaan Waaweyn**:
  - Diinta: Towxiidka, Salaada, Soonka, Xajka
  - Tecnolijiyada: Python, HTML, CSS, PHP, R, Java, C++, SQL
  - Suugaanta: Maahmaahyada, Xikamadaha, Gabayga, Heesaha
- **Quran Audio Player**: 14 Surah iyo 4 Qaari
- **Bogag Caalami**: About, Contact, Terms, Privacy
- **Subscribe Section**: YouTube @filanwaa
- **Social Media**: YouTube, Instagram, Facebook, Twitter
- **Google AdSense Ready**: Ad banner placements
- **Responsive Design**: Mobile, Tablet, Desktop

## Tech Stack

- React 18 + TypeScript
- Vite 5
- React Router DOM 6
- Custom i18n (3 languages)
- FFmpeg.wasm (client-side video encoding)

## Deployment

### Vercel
1. Push this repo to GitHub
2. Import to Vercel
3. Deploy — `vercel.json` included

### GitHub
```bash
git init
git add .
git commit -m "Filanwaa - Diinta, Tecnolijiyada iyo Suugaanta"
git branch -M main
git remote add origin https://github.com/filanwaa/filanwaa.git
git push -u origin main
```

## Development

```bash
npm install
npm run dev
npm run build
```
